# 📝 NoteCollate — 笔记整理

> **零散笔记 → 结构化复习资料**

一个 AI 笔记整理工具。把你的课堂笔记、课件、录音交给他，自动重整为清晰的结构化复习资料。

---

## 它能做什么

| 功能 | 说明 |
|------|------|
| 📖 **整理单节课笔记** | 刚下课的随手拍，重整为结构清晰的课堂笔记 |
| 📚 **合并整学期笔记** | 所有课件和笔记按时间合并为一份完整复习资料 |
| ⭐ **区分重点** | 必考 ⭐⭐⭐、熟悉 ⭐⭐、了解 ⭐，一目了然 |
| 🏃 **考前速览** | 知识图谱 + 考点清单 + 高频问答模板 |
| 💡 **课件精炼** | PPT 要点补上逻辑连接词，读起来顺畅 |
| 📝 **模拟出题** | 根据内容生成基础题、作业变形题 |
| 🎯 **综合复习** | 课件+作业+小测+真题交叉分析，精准定位考点 |

## 三种整理模式

- **课堂笔记整理** — 按课时重整，7 步结构 + 星级标注 + 一句话精炼 + 比较分析
- **期末复习速览** — 考点清单 + 知识图谱 + 简答模板 + 考前最后一页
- **课件精炼** — PPT 要点 → 连贯段落 + 原始要点对照

## 你需要提供什么

**一份原始材料就行，格式不限：**

- ✏️ **文本** — 直接粘贴，或丢 .txt / .md / .docx 文件
- 📊 **课件** — PDF、PPT 都能直接读
- 📸 **图片** — 拍的书页、板书直接拖进窗口

> 给多给少都行，一条随堂笔记也能整理，整学期的材料也能一次性处理。

如果有参考教材、课程大纲、作业、小测、历年真题，一起丢过来效果更好 —— 会自动交叉分析考点频率。

## 快速开始

需要 [Codex](https://codex.ai) 环境，将文件夹放入 Codex 的 skills 目录：

```bash
# macOS / Linux
cp -R note-collate ~/.codex/skills/

# Windows PowerShell
Copy-Item -Recurse note-collate "$env:USERPROFILE\.codex\skills\"
```

完成后在 Codex 中直接说：

> 帮我把这些笔记整理一下

## 适用于

任何以文字笔记为主要学习材料的课程 —— 人文社科、理工农医、商科管理，以及会议记录、项目文档、读书笔记等。

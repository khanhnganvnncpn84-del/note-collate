# 模式 A：课堂笔记整理 — 详细模板

## 适用场景

用户有按课时组织的原始笔记（手写/打字/录音转写），需要重整为结构清晰的章节笔记。

## 输入特征

- 内容按日期或课时组织（"5月21日""第3讲""Lecture 5"）
- 每个课时包含多个主题/概念/理论/人物/方法
- 可能是连续段落（录音转写风格）或已有一定结构
- 可能有教师口头强调的重点、必考点标记

## 输入检测与分割

### 连续文本分割

如果输入是连续段落（录音转写风格），先按以下规则分割：

1. 按日期关键词分割："X月X日""第X讲""Lecture X"
2. 在每个课时内，按核心要素切换分割：
   - 人物切换（人名 + 生卒年）
   - 概念/术语引入
   - 作品/文本讨论
   - 思潮流派转换
3. 识别段落内的结构要素：定义 → 观点 → 例证 → 评价

### 结构化要素提取

分割后提取为标准化条目，每个条目的通用格式：

```
[类型标签] [名称]（[原文/外文名]，[时间/日期]）

**定位**：[一句话定位该要素在整个课程中的角色]

**核心内容**：
1. [要点1]
2. [要点2]
...

**关键引文/例证**（如有）：
- [具体的文本引用或例证]

**一句话总结**：[精炼概括]

**意义/影响**（如有）：
- [对该领域或后续的影响]

**局限性/批评**（如有）：
- [局限1]
```

### 要素类型

常见要素类型：

**人物**（最常见）：
- 定位：历史地位、所属领域
- 代表作：列出作品名（附原文）
- 核心观点：分点列出主张
- 一句话精炼

**概念/术语**：
- 定义：精确的学术定义
- 关键特征：2~4 个区分点
- 相关人物/文本
- 与相似概念的辨析

**作品/文本**：
- 作者 + 年代
- 核心内容/主旨
- 在学科史上的地位/影响
- 关键引文

**思潮流派**：
- 时间范围 + 地域
- 核心主张
- 代表人物
- 与其他流派的区别/联系

**事件/分期**：
- 时间 + 背景
- 关键内容
- 历史意义

## 输出结构（7 步 + 增强元素）

### Step 0: 总体主线（跨课时）

跨越多个课时时，在开头给出"总体脉络"：

```markdown
# 课程名称 - 课堂笔记整理

## 总体脉络

1. [主题1]：[一句话概括]
2. [主题2]：[一句话概括]
```

### Step 1: 本章重点/星级标注

每节课开头列出重点要素，标注重要性：

```markdown
### ⭐⭐⭐ [要素名] — [原因，如"必考简答/论述"]
### ⭐⭐ [要素名] — [原因，如"常考名词解释"]
### ⭐ [要素名] — [原因，如"了解即可"]
```

### Step 2: 核心概念引入

提取本节课的核心概念：

```markdown
- **概念名**（[原文]）：[精确定义]
  - 关键特征 1
  - 关键特征 2
  - 与相似概念的辨析
```

### Step 3: 要素详细笔记

每个核心要素按统一格式展开：

#### 人物类

```markdown
### [人物名]（[外文名]，[生卒年]）⭐⭐⭐

**定位**：[一句话定位]

**代表作品**：
1. 《作品名》（[原文名]）— [简要说明]

**核心观点**：
1. [观点1]
2. [观点2]

**一句话**：[精炼概括]

**意义**：
- [影响]

**局限性**：
- [局限]
```

#### 概念类

```markdown
### [概念名]（[原文名]）⭐⭐

**定义**：[精确定义]

**关键特征**：
1. [特征1]
2. [特征2]

**相关人物/文本**：
- [人物/文本]：[关系说明]

**辨析**：
- vs [相似概念]：[区别]
```

#### 思潮流派类

```markdown
### [流派名]（[原文名]）⭐⭐⭐

**时间**：[时间范围] | **地域**：[地域]

**核心主张**：
1. [主张1]
2. [主张2]

**代表人物**：
- [人物1]：[贡献]
- [人物2]：[贡献]

**与其他流派的关系**：
- [异同]
```

### Step 4: 关键引文/例证

文本分析和引文是关键：

```markdown
**关键引文**：[原文摘录]
- **来源**：[作者/作品/章节]
- **解读**：[该引文说明了什么]
- **考试切入点**：[可能的考题方向]
```

### Step 5: 比较分析（跨要素）

对相关要素进行横向比较：

```markdown
## 比较分析

| 维度 | 要素A | 要素B |
|------|-------|-------|
| 时间 | | |
| 核心主张 | | |
| 代表作 | | |
| 影响 | | |
```

### Step 6: 巩固练习/思考题

从笔记中提取或生成练习题（留白）：

```markdown
1. [题目]（提示：[考察方向]）
```

答案单独放在文档末尾。

### Step 7: 总结复盘

```markdown
## 总结复盘

### 知识脉络
[本课时内容的逻辑梳理]

### 易混淆点
1. [易混点] — [区分方法]

### 前后联系
- 前置知识：[需要先掌握的内容]
- 后续：[以此为基的内容]
```

## 完整合并结构

```
# 课程名称 - 课堂笔记整理

## 总体脉络（跨课时）

## [第一课时]
### ⭐⭐⭐ [重点要素]

### [要素1]
**类型**：[概念/人物/成果/方法]
[按对应类型模板展开]

### [要素2]
...

## [第二课时]
...

## 比较分析
[跨课时的横向比较]

## 习题答案与提示
```
